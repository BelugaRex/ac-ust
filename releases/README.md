# AC-UST 公测版本

此目录存放打包好的 CRX 文件，供公测用户下载安装。

## 安装方法

1. 下载 `.crx` 文件
2. 打开 `chrome://extensions` 或 `edge://extensions`
3. 开启 **开发者模式**（Developer mode）
4. 将 `.crx` 文件拖入浏览器窗口
5. 确认安装

## 版本说明

文件名格式：`ac-ust-vX.Y.Z.crx`

- **v0.x** — 公测版本，功能可能不稳定
- 每个版本均用同一私钥签名，更新时会保留数据
